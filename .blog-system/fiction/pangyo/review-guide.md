# 허공을 미는 사람들 - 리뷰 채점 기준

> 이 문서는 소설 "허공을 미는 사람들"의 장면 리뷰 시 사용하는 채점 기준입니다.
> 5개 항목 각 20점, 총 100점. **95점 이상 통과.**

---

## 채점 항목

### 1. 몰입도 (Immersion) — /20

독자가 장면에 빠져들어 읽는 것을 멈추지 않게 하는 힘.

| 점수 | 기준 |
|------|------|
| 18~20 | 장면을 읽는 동안 현실을 잊는다. 긴장감/감정이 끊기지 않는다. |
| 15~17 | 대체로 몰입되지만, 1~2곳에서 흐름이 약간 끊긴다. |
| 12~14 | 몰입이 간헐적. 중간중간 빠져나오는 느낌. |
| 11 이하 | 몰입이 거의 안 된다. 장면이 산만하거나 지루하다. |

**체크포인트:**
- 장면의 도입이 자연스럽게 독자를 끌어들이는가?
- 긴장감이 적절히 유지되는가? (부서장 면담, 팀원 충돌 등)
- 장면의 마무리에 여운이 있는가?
- 불필요한 설명이 몰입을 깨지 않는가?
- 대화와 서술의 리듬이 읽기 편한가?

---

### 2. 문체 일관성 (Style Consistency) — /20

`writing-styles.md`의 문체 가이드를 얼마나 충실히 따르는가.

| 점수 | 기준 |
|------|------|
| 18~20 | 문체 가이드의 모든 규칙을 자연스럽게 따른다. 의식하지 않아도 톤이 일관된다. |
| 15~17 | 대부분 따르지만, 1~2곳에서 톤이 벗어난다. |
| 12~14 | 부분적으로 따르지만, 톤의 흔들림이 눈에 띈다. |
| 11 이하 | 문체 가이드와 상당히 다르다. |

**체크포인트:**
- 시제: 과거형 서술 + 현재형 내면 독백 교차가 자연스러운가?
- 문장 리듬: 짧은 문장 연쇄, 체언 종결, 단답 반복이 적절히 쓰였는가?
- 감정 표현: 감정을 직접 말하지 않고 사물/행동/타인 반응으로 전달하는가?
- 금지 표현: 감정 형용사 직접 나열, 자기 연민, 과잉 드라마가 없는가?
- 호칭: "부서장님" 통일, 성+직책 규칙을 따르는가?
- 시간 표기: `### [오전 9:31] 박 팀장 - 사무실에서` 형식인가?
- 숫자 표기: "석 달", "여섯 살" 등 규칙대로인가?

---

### 3. 장면 묘사 (Scene Description) — /20

인물의 심리를 짐작하게 하는 환경/사물/공간 묘사의 품질.

| 점수 | 기준 |
|------|------|
| 18~20 | 묘사가 인물 심리와 유기적으로 결합. 읽는 사람이 장면을 '보고 느낀다'. |
| 15~17 | 묘사가 있고 심리와 연결되지만, 일부 장면에서 더 풍부할 수 있다. |
| 12~14 | 묘사가 빈약하거나, 심리와의 연결이 약하다. |
| 11 이하 | 묘사가 거의 없거나 장식적이다. 기존 에피소드 1, 2-1 수준 이하. |

**체크포인트:**
- 공간의 분위기가 인물의 심리 상태를 반영하는가?
  - 예: 식은 국 = 닿지 못하는 돌봄, 꺼진 불 = 고립
- 대화 사이에 적절한 여백이 있는가?
  - 침묵, 신체 반응(손 떨림, 다리 힘 풀림), 사소한 행동(시선 이동, 물건 만지기)
- 사물 묘사가 과하지 않으면서도 효과적인가?
  - 건조한 톤을 유지하면서 심리를 투영
- 반복 모티프(시계 초침, 식은 밥, 흡연실)가 적절히 활용되는가?
  - 같은 모티프 한 장면에서 2회 이하
- **기존 대비 향상**: 에피소드 1, 2-1보다 장면 묘사가 풍부해졌는가?
  - 대화만 나열하지 않고, 대화 전후에 공간과 인물의 상태를 그리는가?

---

### 4. 인물 심리 (Character Psychology) — /20

등장인물의 내면이 설득력 있게 그려지는가.

| 점수 | 기준 |
|------|------|
| 18~20 | 인물의 감정 변화가 자연스럽고, 행동의 동기가 명확하다. 독자가 공감한다. |
| 15~17 | 대체로 자연스럽지만, 1~2곳에서 감정 변화가 급격하거나 설명 부족. |
| 12~14 | 인물의 심리가 평면적이거나 예측 가능하다. |
| 11 이하 | 인물이 설정과 다르게 행동하거나, 심리 묘사가 거의 없다. |

**체크포인트:**
- 화자의 감정 곡선이 인터뷰에서 합의한 대로인가?
- 감정 변화의 전환점이 자연스러운가?
  - 갑자기 화내거나 갑자기 울지 않는가?
- 캐릭터 아크(settings.md)와 일치하는가?
  - 박 팀장: 중간관리자의 무력감, 말 안 통하면 대답 못 하는 습관
  - 김 대리: 실력 있지만 환멸, 책임감에 남아있는 인물
  - 하 과장: 에피소드별 성격 변화 궤적
  - 부서장: 악당이 아님, 여유로운 척 압박
- 타인의 내면을 추측으로만 표현하는가? (`~인 것 같았다`)
- 조직 언어의 이면을 화자가 해석하는 구조가 있는가?

---

### 5. 설정 정합성 (Setting Integrity) — /20

작품 설정(settings.md)과 이전 에피소드와의 일관성.

| 점수 | 기준 |
|------|------|
| 18~20 | 모든 설정이 정확하고, 이전 에피소드와 완벽히 연결된다. |
| 15~17 | 대부분 정확하지만, 사소한 불일치가 1~2개. |
| 12~14 | 설정 오류가 3개 이상이거나, 이전 에피소드와 어긋나는 부분이 있다. |
| 11 이하 | 설정을 무시하거나, 캐릭터가 다른 사람처럼 행동한다. |

**체크포인트:**
- 인물 기본 정보: 나이, 연차, 직책, 가족 구성 정확한가?
- 호칭 규칙: 부서장님 통일, 성+직책, 성만 사용
- 프로젝트 기간: "석 달" 통일 (3개월, 세 달 X)
- 아이들 표현: "아이들은 잠들어 있었다" (애들이 잤다 X)
- 이전 에피소드 사건과의 연속성:
  - 에피소드 1의 사건이 에피소드 2에서 자연스럽게 이어지는가?
  - 인물 간 관계 변화가 일관되는가?
- 시나리오 체크리스트(settings.md의 7번, scenario-flow.md 하단) 항목 반영 여부
- 물리적 설정: 부서장실은 2층 위, 건물 밖 흡연실 등

### 6. 문장 흐름 L1: 문장 연결 (Sentence Connection) — /10

> **별도 병렬 에이전트**가 이 항목만 전담 리뷰한다.

의도 없이 끊어진 짧은 문장들이 자연스럽게 연결되어 있는가.
`writing-styles.md` 14.1 (레벨 1) 규칙 준수 여부를 검사한다.

| 점수 | 기준 |
|------|------|
| 9~10 | 행동 나열, 상황 설명이 자연스럽게 흐른다. 의도적 단절만 존재한다. |
| 7~8 | 대체로 자연스럽지만, 1~3곳에서 불필요한 끊김이 있다. |
| 5~6 | 전보문 스타일의 짧은 문장이 5곳 이상. 흐름이 끊긴다. |
| 4 이하 | 대부분의 서술이 전보문 스타일. 읽기 불편하다. |

**체크포인트:**
- 연속 동작이 한 문장씩 끊어져 있지 않은가? (`~고`, `~서`로 연결해야 할 곳)
- 인과/상황이 별도 문장으로 불필요하게 분리되어 있지 않은가? (`~는데`, `~니`)
- 의도적 단절(충격, 강조, 단답 반복)은 그대로 유지되는가?
- **의도 없이 너무 짧게 끊어치는 문장에는 반드시 감점**

**감점 기준:**
- 의도 없는 3단어 이하 문장이 연속 3개 이상: **-2점**
- `했다. 했다. 했다.` 패턴의 행동 나열: **-1점/건**
- 연속 동작을 개별 문장으로 나열 (`핸드폰을 꺼냈다. 시간을 봤다.`): **-1점/건**

**의도적 단절 vs 의도 없는 끊김 판별법:**
- 의도적: 충격 직후, 감정 폭발, 단답 반복(`예.` `예.`), 강조, 장면 마무리
- 의도 없음: 평서 서술, 일상 동작 묘사, 상황 설명, 회상 나열

---

### 7. 문장 흐름 L2: 서술 확장 (Narrative Depth) — /10

> **별도 병렬 에이전트**가 이 항목만 전담 리뷰한다.

내면 독백, 회상, 감정 표현에서 충분한 깊이와 확장이 있는가.
`writing-styles.md` 14.2 (레벨 2) 규칙 준수 여부를 검사한다.

| 점수 | 기준 |
|------|------|
| 9~10 | 생각의 꼬리물기, 공간 투영, 회상 서사화, WHY 탐색이 자연스럽게 구현된다. |
| 7~8 | 대체로 깊이가 있지만, 1~2곳에서 결론만 서술하고 끝난다. |
| 5~6 | 내면 독백이 얕다. WHY 탐색이 부족하고 결론에서 끊긴다. |
| 4 이하 | 감정/생각이 한 문장으로 끝나고 확장이 거의 없다. |

**체크포인트:**
- **Pattern A (회상 서사화):** 과거 사건이 전보문이 아닌 시간 흐름에 따른 서사로 전개되는가?
- **Pattern B (내면 심층):** 감정의 결론만 있고 WHY가 빠져있지 않은가?
- **Pattern C (공간 투영):** 감정이 장소/사물/환경과 연결되어 무게를 갖는가?
- **Pattern D (꼬리물기):** 생각이 결론에서 멈추지 않고 추론/상상/의문으로 이어지는가?

**감점 기준:**
- 내면 독백이 결론 1문장으로 끝나는 곳이 5곳 이상: **-2점**
- 회상이 전보문 나열로 처리된 곳: **-1점/건**
- 핵심 감정 장면에서 공간/사물과의 연결이 전혀 없는 곳: **-1점/건**

---

## 통과 기준

> 기존 5개 항목(각 /20) + 문장 흐름 L1(/10) + L2(/10) = **총 120점**
> L1, L2 리뷰는 기존 5개 항목 리뷰와 **병렬로 진행**한다.

| 총점 | 판정 | 다음 행동 |
|------|------|----------|
| 114~120 | **PASS** | 승인 가능. 다음 장면으로 진행. |
| 108~113 | **NEEDS MINOR REVISION** | 특정 항목만 수정 후 재리뷰. |
| 96~107 | **NEEDS REVISION** | 여러 항목 수정 필요. 자동 수정 시도. |
| 95 이하 | **REWRITE** | 장면 전체 재작성 권장. 인터뷰 재진행 고려. |
