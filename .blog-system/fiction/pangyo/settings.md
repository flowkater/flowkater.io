# 판교 전세에 IT대기업 다니는 박 팀장 이야기

## 작품 설정 및 가이드 문서

---

## 1. 작품 개요

### 제목
**판교 전세에 IT대기업 다니는 박 팀장 이야기**
- "서울 자가에 대기업 다니는 김 부장 이야기" 패러디
- 겉으로는 괜찮아 보이는 조건, 실상은 다른 아이러니

### 주제
**"중간에 낀 사람들의 무력감"**
- 조직 내 위와 아래 사이에 끼어 있는 사람들의 이야기
- 누구도 악의를 가지고 있지 않지만, 시스템 안에서 모두가 피해자가 됨
- 특정 인물을 악당으로 그리기보다 구조가 만들어내는 고통을 보여줌
- + 고객이 없는 프로젝트, 승리하지 못하는 조직의 필연적 불행을 다룸

### 시점 구조
- 팀장(박 차장) 시점과 팀원(김 대리) 시점 교차
- 같은 상황을 다른 각도에서 조명
- 시간 표기 활용: [오전 9:12], [오후 7:34] 형식

### 톤
- 건조하면서도 내면을 드러내는 문체
- 담담함 아래에 있는 분노와 체념
- 직장인이라면 공감할 수 있는 상황들

## 3. 에피소드 구조

### 전체 타임라인

```
에피소드 1: 프로젝트 막바지 (석 달째)
    ↓
에피소드 2: 프로젝트 종료 직후
    ↓
에피소드 3: 두 달 뒤 (판매 실패, 균열, 기획서 승인)
    ↓
에피소드 4: 석 달 뒤 (희망 → 좌절 → 퇴사)
```

---

### 에피소드 1: 위기 발생

**시점:** 프로젝트 석 달째 (주말 포함 야근 지속)

**장면 구성:**
1. [팀장] 부서장실 호출 - AI 기능 추가 요구
2. [팀원-김 대리] 사무실 - 복사붙여넣기 자괴감, 기술문화 실망
3. [팀장] 집 - 부서장 밤 전화 (사무실에 아무도 없다)

**핵심 사건:**
- 제외하기로 한 AI 통합 기능 다시 추가 지시
- 팀원들 불만 극심하지만 일정 내 완료 가능할 것 같았음
- 부서장의 "푸시하니까 되네요" 발언
- 팀장이 팀원 편들다가 "더 열심히 해보겠습니다"로 굴복

**분량:** 기준 분량 (1배)

---

### 에피소드 2: 허무한 성공

**시점:** 프로젝트 종료 직후

**장면 구성:**
1. [팀원-김 대리] 사무실 - 끝났는데 허전함, 보상 기대
2. [팀장] 부서장실 - "잘했다" 칭찬, 보상 구체화 문의 → 부서장이 "성과급"으로 리프레이밍 → "판매 성과 나오면"
3. [팀원-김 대리] 회의실 - 팀장과 1:1, 리프레이밍된 답변 전달
4. [팀장] 집 - 아내와 대화, 불안한 희망

**핵심 사건:**
- 대행업체 극찬 "시장에서 대박"
- 다들 기대했던 보상 → 부서장이 "성과급"으로 리프레이밍 → "판매 성과 나와야"
- 석 달 고생의 대가가 "조금만 기다려"

**분량:** 기준 분량 (1배)

---

### 에피소드 3: 균열

**시점:** 프로젝트 종료 후 두 달 뒤

**장면 구성:**
1. [팀원-김 대리] 사무실 - 분위기 싸함, 성과급 여전히 없음
2. [팀원-김 대리] 사무실 - 판매 실패 소식 (팀장 통해 전달)
3. [팀장] 부서장실 - 부서장 분노, 추가 개발 요구
4. [팀원-김 대리] 사무실 - 팀장 발표, 하 과장 충돌
5. [팀원-김 대리] 사무실 - 팀장-하 과장 회의실 싸움 관찰
6. [팀원-김 대리] 회의실 - 퇴사 선언, 팀장이 붙잡음
7. [팀장] 집 서재 - 새벽, 기획서 작성
8. [팀장] 부서장실 - 기획서 보고, **승인**

**핵심 사건:**
- 고객사 악평 "돈 받고 써야 할 정도"
- 성과급 완전히 물 건너감
- 하 과장과 충돌 → "니가 진짜 그렇게 열심히 했냐고"
- 김 대리 퇴사 선언 → 팀장이 "바꿔보겠다" 약속
- **다이렉트 프로젝트 기획서 승인** (이전 프로젝트 추가 개발 먼저 마무리 조건)

**중요:** 악평 전달 방식 - 천장에서 소리 들리는 것(X) → 팀장이 내려와서 전달 또는 사내 메신저/소문으로

**분량:** 기준 분량 1.5~2배

---

### 에피소드 4: 결말

**시점:** 에피소드 3 이후 석 달 뒤 (다이렉트 프로젝트 두 달째 진행 중)

**장면 구성:**
1. [팀장] 사무실 - 요즘 출근이 싫지 않다, 프로젝트 잘 진행 중
2. [팀원-김 대리] 사무실 - 분위기 나아짐, 하지만 완전하지 않음
3. [팀장] 부서장실 - 갑자기 호출, **전면 재개발 지시**, 다이렉트 보류
4. [팀장] 자리 - 결심, 인사팀 메시지
5. [팀장] 지하 주차장 - 퇴사, 눈물 (억울해서)
6. [팀원-김 대리] 사무실 - 팀장 빈자리 발견, 사원증만 남음
7. [팀원들] 부서장실 앞 - 전원 호출
8. [팀원-김 대리] 부서장실 - 팀장 퇴사 통보, 다이렉트 보류, 재개발 지시
9. [팀원-김 대리] 퇴근길 - 남겨진 자의 혼란, "내일도 출근해야 한다"

**핵심 사건:**
- "아 그거 아직도 하고 있었어요?" - 다이렉트 프로젝트 무시
- 300억 계약 운운하며 전면 재개발 지시
- 팀장 즉각 퇴사 결심 → 인사팀 메시지 → 사원증 남기고 퇴장
- 팀원들 부서장실 호출로 마무리 (열린 결말)

**분량:** 기준 분량 4배 (가장 김)

## 5. 문체 가이드

### 5.1 기본 원칙

- 건조하면서도 내면을 드러내는 톤
- 담담함 아래의 분노와 체념
- 과도한 감정 표현 자제
- 행동과 상황으로 감정 전달

### 5.2 피해야 할 것

| 항목 | 내용 |
|------|------|
| 문장 종결 반복 | "~했다. ~했다. ~했다." 단조로움 피하기 |
| 내면 묘사 과잉 | 같은 감정 여러 번 반복 설명 X |
| 시제 혼용 | 과거형으로 통일 |
| 과도한 비유 | 문학적 표현 절제 |

### 5.3 권장 표현

| 기존 | 수정 |
|------|------|
| 애들이 잤다 | 아이들은 잠들어 있었다 |
| 애들 자는 걸 본 게 | 잠든 얼굴을 본 게 |
| 한 달 | 석 달 |
| 3개월 / 세 달 | 석 달 |

### 5.4 시간 표기

```
[오전 9:31] 팀장 - 사무실에서
[오후 4:15] 팀원 - 회의실에서
```

- 각 장면 시작에 시간 표기
- 시간 흐름으로 긴장감/급박함 표현
- 팀장 퇴사 장면: 30분 내 모든 일이 진행됨을 시간으로 보여주기
