## 전체 타임라인 요약

```
에피소드 1: 프로젝트 막바지 (석 달째) ✅ 완성
    ↓
에피소드 2: 프로젝트 종료 직후 (약 1주일간)
  ├─ 2-1: 토요일, 처가댁(박팀장 시점), 사무실 (김대리 시점) ✅ 완성
  ├─ 2-2: 부서장실 결과보고 (박 팀장 시점), 보상 리프레이밍 통보 + 퇴근 (김 대리 시점)
```

## 에피소드 2-2: 결과보고

**시점:** 수요일 오전 / **화자:** 박 팀장

### 장면 흐름

**① [수요일, 오전 9:50] 박 팀장 - 부서장실로 올라감**
- 릴리즈 후 첫 공식 보고
- 올라가기 전 심호흡 (에피소드 1과 같은 습관)
- 마음속: 대행업체 반응 좋았으니 괜찮겠지. 챙겨주신다고 한 거 구체적으로 물어봐야 하는데.

**② [수요일, 오전 10:05] 부서장실 - 보고**
- 부서장: "팀장님, 고생했어요." (처음으로 밝은 표정)
- 대행업체 피드백 전달: "시장에서 대박 날 것 같다", "기능도 잘 갖춰져 있고 UI도 깔끔하다"
- 부서장: "팀장님이 팀원들 잘 이끌어서 가능했던 것 같아요."
  - 칭찬인데 이상함 — 밤 10시에 "사무실에 아무도 없다"고 했던 사람
- 팀장이 조심스럽게 "챙겨주신다고 한 건" 구체적으로 물어봄 (금액, 유급 휴가 기간 등 (연차는 당연히 쓸 수 있다고 생각해서 안 물어봄))

**③ 보상 약속의 리프레이밍**
- 부서장: "성과급 말하는 거예요?" ← 팀장이 꺼내지 않은 단어를 부서장이 먼저 명명
- "성과가 있어야 성과급이에요."
- "개발 완료는 성과가 아니에요. 판매가 성과지."
- "판매 성과가 나오면 그때 논의하죠."
- 언제? "빠르면 한두 달, 길면 분기 이상."
- 팀장 내면: 넉넉하게 챙겨주겠다고 했을 때 성과급이라고는 안 하셨다. 우리가 그렇게 들은 거다.

**④ 추가 통보**
- 부서장: "그리고 연차도 일단 쓰지 말라고 해요."
- 대행업체에서 수정 요청 들어올 수 있음. 대기 필요.
- "오늘은 칼퇴시켜요." — 베푸는 듯한 톤
- 부서장: "아, 그리고 다음 프로젝트 준비도 좀 해주세요."
  - 칭찬 직후 바로 다음 요구사항 (캐릭터 설정 그대로)

**⑤ 부서장실 나옴**
- 복도에 서서 멍. 다리 후들거림.
- 팀원들한테 뭐라고 말하지.
- 건물 밖 흡연실로 감 (담배 안 피움. 그냥 서 있음.)

### 핵심 포인트
- 부서장은 "성과급"이라고 한 적 없음 → "넉넉하게 챙겨줄게"라는 애매한 약속
- 팀장과 팀원들이 알아서 "보너스 + 유급 휴가"로 해석 (밀린 연차는 당연) → 부서장에게 plausible deniability 제공
- 팀장이 구체적으로 물어보자 부서장이 "성과급 말하는 거예요?"로 리프레이밍 → 성과 핑계
- "성과가 있어야 성과급" — 논리적으로 맞지만, 애초에 성과급이라고 한 적 없는 사람의 말
- 칭찬 → 거절 → 추가 요구 (부서장 캐릭터 패턴)
- 팀장의 무력감: 위에서 받은 걸 아래로 전달해야 하는 역할

**시점:** 수요일 오전~퇴근 / **화자:** 김 대리

### 장면 흐름

**① [수요일, 오전 10:40] 김 대리 - 사무실**
- 팀장님이 부서장실에서 내려옴. 표정 묘함. 눈이 멍함.
- "다들 잠깐." (에피소드 1과 같은 패턴 — 이 말이 나오면 안 좋은 소식)

**② 팀장 전달**
- "부서장님께서 잘했다고. 대행업체 반응도 좋다고."
- 정 대리 밝아짐. 하 과장 "그럼 이제 챙겨주신다는 거 나오는 거 아니에요?"
- 팀장 잠깐 침묵 후: **"챙겨준다는 게 성과급이었대. 판매 성과 나와야 책정 가능하대."**
- 사무실 정적.

**③ 하 과장 반응 (긴장 고조, 아직 폭발은 아님)**
- "토요일에 부서장님이 직접 넉넉하게 챙겨준다고 하셨잖아요."
- "저도 개인 일정 취소하고 일요일에 나왔어요."
- "그랬는데 언제인지 모른다고요?"
- 팀장 대답 못 함. 침묵.
- 하 과장 더 따지려다 **참음**: "…됐어요." (이 한마디로 마무리)
  - 아직 폭발하지 않음 → 에피소드 3에서 폭발하기 위한 긴장 축적

**④ 추가 통보: 휴가 보류 + 수정 대기**
- "대행업체 수정 요청 올 수 있으니 대기하래."
- "휴가는 수정 끝나고 쓰래."
- 석 달 고생한 대가가: 보상은 성과급으로 리프레이밍되어 미정 + 휴가 보류 + 수정 대기
- 하 과장 코웃음. 이 과장은 아무 반응 없음 ("뭐 그렇지" 표정).
- 윤 사원은 눈치만 봄.

**⑤ 칼퇴 — "오늘은 일찍 가"**
- 팀장: "오늘은 부서장님이 칼퇴하라고 하셨어. 일찍 가."
- 여섯 시. 다들 일어남. 하 과장이 제일 먼저 나감. 가방 챙기고 인사도 안 함.
- 이 과장: 조용히 나감.
- 정 대리: "팀장님, 수고하셨습니다." (아직 긍정적)
- 윤 사원: "수고하셨습니다." 기계적으로.

**⑥ [저녁 7시쯤] 김 대리 - 퇴근길**
- 평소보다 일찍 나왔는데 허전함.
- 석 달 동안 매일 밤 10~11시에 나갔는데, 7시에 나오니 어색함.
- 지하철 (원룸 자취 → 에피소드 4 마지막에 거주지 재언급)
- 개발자로서의 자괴감 회상:
  - GitHub 잔디 텅텅 빔 (개인 계정)
  - 예전에는 퇴근 후에 사이드 프로젝트도 했는데, 지금은 침대에 누워 유튜브 보다 잠듦
  - 이직해야 하나. 1년도 안 됐는데.
- 마무리: **"내일도 출근해야 한다. 대기하라고 했다. 수정 요청이 올 수 있다고. 성과급은 언제인지 모른다고. 석 달 고생한 대가가 그거다."**
- 에피소드 1 엔딩과 호응: **"내일은 온다. 준비 같은 건 상관없이."**

### 핵심 포인트
- 부서장의 애매한 약속("넉넉하게 챙겨줄게")이 팀원들의 기대를 키움 → 리프레이밍으로 배신
- 하 과장의 "…됐어요." → 분노를 참고 있는 상태. 에피소드 3 폭발의 복선.
- 석 달 야근의 대가 = 칼퇴 하루. 이 한 줄의 아이러니.
- 김 대리의 환멸 심화: 기술 성장 정체 + 보상 없음 + 고객도 없는 프로젝트
- 에피소드 2 전체 톤: "끝났는데 뭐가 남은 거지" — 허무한 성공

## 에피소드 3: 균열

**시점:** 프로젝트 종료 후 두 달 뒤 / **분량:** 기준 1.5~2배

### 전체 구조

```
두 달간 시간 경과:
- 수정 요청 들어옴 → 잡일 처리
- 성과급 여전히 없음
- 팀 분위기 점점 싸해짐
- 하 과장: 칼퇴 시작, 농담 안 함
- 이 과장: 더 무기력해짐
- 김 대리: 이직 사이트 둘러봄
```

### 장면 흐름

**① [오전 9:15] 김 대리 - 사무실 (두 달 뒤의 일상)**
- 분위기 묘사: 석 달 전 프로젝트 때보다 조용하지만 다른 종류의 침묵
- 프로젝트 때는 바빠서 조용했고, 지금은 의욕이 없어서 조용함
- 하 과장 변화: 예전에는 농담도 하고 팀 분위기 메이커였는데, 지금은 필요한 말만 함. 칼퇴 고수.
- 이 과장: "그냥 그래"가 입버릇. 눈에 생기 없음.
- 성과급 얘기 아무도 안 꺼냄. 금기어가 됨.
- 김 대리: 점심시간에 슬쩍 이직 사이트 봄. 죄책감 반, 해방감 반.

**② [오전 11:00] 판매 실패 소식 전달**
- 팀장이 부서장실에서 내려옴. 표정이 아예 없음. 하얗게 질려 있음.
- "다들 잠깐." (세 번째 등장하는 이 패턴)
- **전달 방식:** 팀장이 직접 내려와서 전달 (천장에서 소리 들리는 거 아님)
- 고객사 평가: **"돈 받고 써야 할 정도는 아니다"** — 완곡하지만 치명적
- 대행업체: 판매 중단. 추가 개발 없이는 영업 불가.
- 성과급: **완전히 물 건너감** (판매가 없으니 성과도 없음)

**③ [오후 2:00] 박 팀장 - 부서장실 호출**
- 부서장 분노 (하지만 여유로운 척 유지하면서 압박)
- "팀장님, 이게 팀장님이 자신 있다고 했던 결과예요?"
- 추가 개발 요구: 고객사 피드백 반영해서 고쳐라
- "이거 안 고치면 다음은 없어요."
- 팀장: "팀원들이 지쳐 있습니다. 석 달 야근하고, 성과급도 안 나왔고…"
- 부서장: **"그건 팀장님이 관리할 문제 아니에요?"**
- 팀장 굴복. 또.

**④ [오후 3:30] 김 대리 시점 - 사무실, 팀장 발표**
- 팀장 내려옴. 전체 공유: "추가 개발 들어간다."
- **하 과장 충돌 시작:**
  - "잠깐만요. 아니, 성과급도 안 나왔는데 추가 개발이요?"
  - "석 달 야근하고, 주말에도 나오고, 그 대가가 추가 개발이요?"
  - 점점 목소리 올라감
  - 다른 팀원들 굳어짐

**⑤ [오후 3:45] 김 대리 시점 - 팀장-하 과장 회의실 충돌 (관찰자 시점)**
- 팀장이 하 과장을 회의실로 부름
- 회의실 문 닫혀 있지만 소리가 새어나옴 (유리벽이라 표정 보임)
- 하 과장: (추정) 성과급 약속, 일요일 출근, 개인 일정 취소 건 따짐
- 팀장: 처음에는 달래다가 → 한계에 달함
- **팀장 반말 전환 (격앙):** **"니가 진짜 그렇게 열심히 했냐고"**
  - 이 한마디에 회의실 안 정적
  - 팀장 본인도 후회하는 표정
  - 하 과장: 표정이 굳음. 뭔가가 꺼짐.
- 김 대리가 보는 것: 회의실 유리 너머로 두 사람의 표정 변화
- 하 과장 나옴. 자리로 돌아가서 가방 쌈. **칼퇴.** 이날 이후로 매일 칼퇴.
- 하 과장 성격 변화의 결정적 순간.

**⑥ [오후 5:30] 김 대리 - 회의실, 퇴사 선언**
- 김 대리가 팀장에게 1:1 요청
- "팀장님, 저 퇴사하려고요."
- 팀장 멈춤. 잠깐 침묵.
- 김 대리 이유: "여기서 성장할 수 있을 거라고 생각했는데, 1년 지나보니 아닌 것 같습니다."
  - **"연봉 덜 받더라도, 야근 더 하더라도, 의미 있는 일을 하고 싶습니다."**
  - 돈 문제가 아님. 고객도 없고 방향도 없는 일을 반복하는 게 문제.
- 팀장의 붙잡기:
  - 처음에는 "생각 좀 더 해봐" 식으로 나옴
  - 그러다 진심으로: **"내가 바꿔볼게. 조금만 기다려줘."**
  - 뭔가 계획이 있는 듯한 눈빛. 구체적으로 말하진 않음.
  - "이번 추가 개발 끝나면, 다음에는 제대로 된 프로젝트를 제안해볼 거야."
- 김 대리: 반신반의. 하지만 팀장 눈에서 처음으로 확신 비슷한 걸 봄.
- "…한 달만 더 보겠습니다." (완전한 약속은 아님)

**⑦ [새벽 1:30] 박 팀장 - 집 서재**
- 아내와 아이들 잠든 후.
- 노트북 앞에 앉아 기획서 작성.
- **"다이렉트 프로젝트"** — 직접 고객을 만나서 니즈를 파악하고, 자체적으로 기획-개발하는 프로젝트
  - 지금까지의 문제: 대행업체 거쳐서 요구사항 전달 → 실제 사용자 모름 → 엉뚱한 걸 만듦
  - 해결: 직접 고객 접점을 만들자. 작은 규모로 시작.
- 팀장의 동기: 김 대리한테 한 약속 + 하 과장 충돌의 충격 + 8년간의 답답함
- 스타트업 하던 때 떠올림: 그때는 내가 결정하면 됐다. 망해도 내 책임이었다.
- 기획서 다듬고 또 다듬음. 새벽 3시.
- 아내가 서재에 와서 "자야지" 함. "응, 금방 잘게."
- 잠들기 전 생각: 이게 승인되면 뭔가 바뀔 수 있을까.

**⑧ [며칠 뒤, 오후 4:00] 박 팀장 - 부서장실, 기획서 보고**
- 추가 개발 진행하면서 틈틈이 기획서 보완
- 부서장에게 보고: "이번 추가 개발 마무리되면, 이 프로젝트를 제안드리고 싶습니다."
- 부서장 반응:
  - 처음: 시큰둥. "이걸 왜?"
  - 기획서 넘기며 읽음. 점점 표정 변화.
  - "직접 고객 접점?" "자체 기획?"
  - 부서장 입장에서: 임원 승진에 새로운 성과가 필요함. 대행업체 통한 프로젝트는 실패. 새 카드가 필요.
- **승인.** 단, 조건: **"이번 추가 개발 먼저 깔끔하게 마무리하세요."**
- 팀장 처음으로 부서장실 나오면서 표정 밝음.
- 이것이 에피소드 3의 마지막 장면. 어둠 속의 한 줄기 빛.

### 핵심 포인트
- 판매 실패 → 추가 개발 → 하 과장 충돌 → 김 대리 퇴사 선언 → 기획서 승인
- 하 과장 변화의 **결정적 순간**: "니가 진짜 그렇게 열심히 했냐고"
- 김 대리 퇴사 → 팀장의 각성 촉매
- 기획서 승인 = 에피소드 4의 "희망" 파트 세팅
- 분량 1.5~2배 (약 5,000~6,000자)

## 에피소드 4: 결말 (희망 → 좌절 → 퇴사)

**시점:** 에피소드 3 이후 석 달 뒤 (다이렉트 프로젝트 두 달째 진행 중) / **분량:** 기준 4배 (약 12,000~16,000자)

### 전체 구조

```
[전반부: 희망] — 느긋하고 따뜻한 톤
  장면 1~2: 달라진 분위기, 다이렉트 프로젝트 잘 진행 중
    ↓
[전환점: 좌절] — 급격한 톤 전환
  장면 3: 부서장 호출, 전면 재개발 지시, 다이렉트 보류
    ↓
[후반부: 퇴사와 남겨진 자들] — 빠른 시간 전개 (30분 내)
  장면 4~9: 팀장 퇴사 → 팀원들의 충격 → 열린 결말
```
