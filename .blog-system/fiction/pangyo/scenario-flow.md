### 에피소드 1: 석 달째 ✅

**화자:** 박 팀장 시점 / 김 대리 시점 교차
**시간대:** 프로젝트 석 달째의 어느 하루

#### 확정된 장면 흐름

**① [오전 10:17] 박 팀장 - 부서장실**
- 부서장실 앞 심호흡 습관 첫 등장: "이 문을 열면 다른 사람이 되어야 한다"
- AI 통합 기능 재추가 요구 (이전에 제외하기로 한 것)
- 부서장: "별일 없지요?" → "AI 기능과 통합 기능, 이번 기한에 같이 넣어줄 수 있을까요?"
- 팀장 반발: "그건 제외하기로 했습니다" → "팀원들은 지금도 이미 열심히 하고 있습니다"
- 부서장: "그래요?" (의미심장한 한마디)
- 팀장 항복. 복도에서 다리에 힘 풀림

**② 박 팀장 - 엘리베이터 → 흡연실**
- 지하 1층 야외 흡연실로 내려감 (담배 안 피움, 그냥 서 있음)
- 5년 전 끊은 담배 습관의 잔상
- "팀원들에게 뭐라고 말하지" — 자책

**③ [오후 3:24] 김 대리 - 사무실**
- 팀장님 내려옴 (눈 충혈, 담배 냄새)
- "그 AI 통합 기능 있잖아. 그거 다시 넣어야 할 것 같아."
- 하 과장: "회의에서 빼기로 했잖아. 회의록에도 있고, 메일로도 공유했잖아."
- 김 대리: "일정은요?" → 팀장: "똑같아."
- 팀원 대부분 오후 7시에 나감 (평소보다 이름), 김 대리만 남음
- 하 과장: "씨발" (욕 나옴)

**④ [오후 10:47] 박 팀장 - 집에서**
- 야근 후 집 도착 (30km, 1시간), 아내가 미역국 차려둠
- 부서장 밤 전화: "팀장님, 지금 어디에요?" (인사 없이)
- "열 시쯤 사무실 확인했는데 아무도 없더라"
- "팀원들이 헌신하고 있냐"
- 팀장 내면: "팽팽하게 당겨져 있던 줄이 툭 끊어지는 느낌"
- "내일 아침에 얘기하시죠. 출근하시면 저한테 먼저 오세요."
- 마무리: "여섯 시간 후면 알람이 울린다. 일곱 시간 후면 다시 그 건물 앞에 선다."

#### EP1에서 확립된 핵심 설정

| 항목 | 확정 내용 |
|------|-----------|
| 프로젝트 기간 | 석 달 (주말 출근 포함) |
| 야근 패턴 | 매일 밤 10~11시, 늦으면 새벽 3시 |
| AI 기능 | 제외했다가 부서장 요구로 재추가 (일정 변동 없음) |
| 대행업체 | 소프트웨어 판매 대행, 직접 고객 접점 없음 |
| 코드 리뷰/테스트 | 공식 정책만 존재, 실제 운영 X |
| 커밋 메시지 | "Fix bug", "asdf" 등 무의미해짐 |
| 부서장 습관 | 모니터/서류 보며 말함, 중요할 때만 직시, 안경 너머로 쳐다봄 |
| 박 팀장 습관 | 부서장실 앞 심호흡, 말 안 통하면 대답 안 함, 담배 5년 전 끊음 |
| 박 팀장 신체 | 하루 삼각김밥 2개, 운전 중 졸음, 현기증 |
| 김 대리 전직 | 스타트업 경험, 코드 리뷰 칭찬, 오픈소스 PR |
| 김 대리 거주 | 원룸 자취 (편의점 5분, 계단 있는 골목) |

#### EP1에서 심은 복선

- 부서장의 밤 전화 → "아무도 없더라" (감시)
- 팀장의 "줄이 끊어지는 느낌" → 정신적 한계
- 하 과장의 불만 표출 → EP3 폭발의 씨앗
- 김 대리의 환멸 → 이직 고민의 시작
- 기술 부채 → 대행업체 수정 요청의 원인

### 에피소드 2-1: 허무한 성공 - 1 ✅

**화자:** 박 팀장 시점 (토요일 처가댁) / 김 대리 시점 (수요일 아침)
**시간대:** 토요일 → 일요일(테스트) → 수요일(릴리즈 다음 날)

#### 확정된 장면 흐름

**① [토요일, 오후 2:47] 박 팀장 - 처가댁**
- 장모님 생신, 아이들이 케이크에 초 꽂는 중
- 부서장 전화 (오늘만 네 번째)
- 1차: API 응답 시간 3초 이상 이슈 확인
- 2차: "담당자 전화 안 받아요?" 팀 관리 지적
- 3차: "대행업체에서 내일 테스트하자고 연락 왔어요. 팀원들 다 나와야 할 것 같아요."
- 부서장 톤 전환(부드러워짐): **"이번 프로젝트 끝나면요, 팀원들한테 넉넉하게 해줄게요. 고생한 만큼 보답할 테니까."**
- 4차: "하 과장 연락됐어요?"

**② 박 팀장 - 하 과장과의 전화**
- 1차 전화: 하 과장 거절 ("한 달 전부터 잡아둔 일정")
- 팀장: "…알았어. 어쩔 수 없지."
- 2차 전화: 부서장 지시로 재요청
- **하 과장 거절 유지** (EP2-2에서 확정: "하 과장은 안 나왔다")

**③ 박 팀장 - 처가댁 마무리**
- 아이들: 큰애가 다리 안음, 작은애 "아빠, 놀아줘"
- 딸이 방문 열고 아빠 표정 봄 → 말이 안 나옴 → 문 닫고 천천히 걸어감
- 아내에게: "이번에는 좀 쉴 수 있을 것 같다"
- 팀 단톡: "부서장님 말씀으로는 프로젝트 끝나면 넉넉하게 챙겨주신다고 합니다."

**④ [수요일, 오전 9:31] 김 대리 - 사무실 (릴리즈 다음 날)**
- 최종 릴리즈: 어제 저녁 7시 완료 ("Final release v1.0.0")
- 석 달간의 회고: 복사 붙여넣기, 커밋 메시지 퇴행, 기술 부채 축적
- 일요일 테스트 회상: API 응답 시간 문제 → 김 대리 밤새 디버깅 → 인덱스 추가로 해결 (3초→0.3초)
- **하 과장은 일요일에 안 나옴** (확정)
- 팀원들의 기대: 보너스, 유급 휴가, 밀린 연차

#### EP2-1에서 확립된 핵심 설정

| 항목 | 확정 내용 |
|------|-----------|
| 부서장의 약속 | **"넉넉하게 해줄게요. 고생한 만큼 보답할 테니까."** (의도적으로 모호) |
| 하 과장 일요일 | **안 나옴** (거절, 확정) |
| 일요일 참석자 | 박 팀장, 김 대리, 이 과장, 정 대리, 윤 사원 + 대행업체 |
| API 이슈 해결 | 김 대리가 밤새 디버깅, 인덱스 미설정이 원인 (하 과장 모듈) |
| 릴리즈 시점 | 화요일 저녁 7시 |
| 팀 단톡 메시지 | "넉넉하게 챙겨주신다고 합니다" (구체적 금액/휴가 언급 없음) |
| 팀원 기대 | 하 과장: 휴가 / 이 과장: 유급 휴가 / 정 대리: 보너스 / 윤 사원: 눈치 |

#### EP2-1에서 심은 복선

- **모호한 약속**: "넉넉하게" = 각자 해석 → EP2-2에서 리프레이밍
- **하 과장의 거절**: 일요일 안 나온 유일한 사람 → EP3에서 불만의 근거 변경 필요
- **김 대리의 밤샘**: 기술력 증명 vs. 보상 없음의 괴리
- **팀장의 희망회로**: "이번에는 쉴 수 있을 것 같다" → EP2-2에서 붕괴

### 에피소드 2-2: 허무한 성공 - 2 ✅

**화자:** 박 팀장 시점 (부서장실~흡연실) / 김 대리 시점 (사무실~퇴근길)
**시간대:** 수요일 오전 9:47 ~ 저녁 8시쯤

#### 확정된 장면 흐름

**① [수요일, 오전 9:47] 박 팀장 - 부서장실**
- 심호흡 습관 재등장
- 부서장 "앉아요" (평소와 다름 — 서서 보고가 패턴)
- 칭찬: "수고했어요", "대박 날 거라고"
- 팀장이 먼저 물음: "챙겨주신다고 한 건요"
- **부서장: "성과급 말하는 거예요?"** ← 팀장이 꺼내지 않은 단어
- "판매 성과가 나와야 책정할 수 있어요"
- "빠르면 두세 달? 늦으면 더 걸릴 수도 있고."
- **"성과가 있어야 성과급 아니에요?"** — 논리적으로 반박 불가
- 연차: "일단 쓰지 말라고 해요" → 대행업체 수정 요청 대기
- "길어야 일주일이면 될 거예요"
- **칼퇴 허락**: "오늘은 그래도 일찍 보내줘요. 칼퇴시키고. 내일부터 대기니까."
- 칭찬 → 거절 → 추가 요구 (부서장 캐릭터 패턴 확립)

**② 박 팀장 - 복도 → 엘리베이터 → 흡연실**
- 형광등 깜빡임 (반복 모티프)
- 엘리베이터 스테인리스 문에 얼굴 비침: "누구 얼굴인지 모르겠다"
- 담배 욕구: "입술이 마른 느낌. 필터를 무는 감촉."
- "담배를 피우러 가는 게 아니다. 그냥 거기 서 있으려고."
- 흡연실: "건물 옆 구석. 재떨이 하나. 꽁초가 수북."
- 토요일 회상: 장모님 생신, 케이크, 부서장 전화, 아내에게 한 말
- **"하 과장 빼고 다들 나왔다. 일요일에."** (확정)
- "일 분만 더" (시간 연기의 반복)
- 열 시 삼 분 → 올라가야 한다

**③ 박 팀장 - 엘리베이터 → 사무실 복귀**
- 심호흡 재등장: "이번에는 부서장이 아니라 팀원들 앞에서"
- 유리 너머로 팀원들 보임 (김 대리: 모니터, 하 과장: 커피, 정 대리: 뭔가 적음)
- "표정을 만들었다. 어떤 표정을 짓고 있는지는 모르겠다."
- "좋은 소식을 먼저 말하자."

**④ [수요일, 오전 10:15] 김 대리 - 팀장님 발표**
- 팀장님 관찰: "발을 끌듯이 걸었다", "시선이 나를 통과한 느낌"
- **담배 냄새**: "희미하게 담배 냄새가 났다" → 의아하되 의미 부여 없이 패싱
- "다들 잠깐" (이 패턴이 나오면 좋은 소식인 적 없었다)
- 좋은 소식: "부서장님께서 잘하셨다고. 대행업체 반응도 좋대."
- 정 대리 밝아짐, 하 과장: "챙겨주신다는 거 나오는 거예요?"
- **"챙겨준다는 게… 성과급이었대."**
- 침묵. 에어컨 소리.

**⑤ 김 대리 - 하 과장 반응 관찰**
- 하 과장: "토요일에 부서장님이 직접 넉넉하게 챙겨주신다고 하셨잖아요."
- **"저한테도 일요일에 나오라고 전화하셨잖아요."** (하 과장은 거절하고 안 나옴)
- "그랬는데 언제인지 모른다고요?"
- 팀장 대답 안 함 (대답하지 않는 습관)
- **"…됐어요."** (한마디로 마무리, EP3 폭발의 복선)
- 김 대리 내면: **"나는 처음부터 회의적이었다. 사실 크게 중요하지 않다."**
- 추가 통보: 수정 요청 대기, 연차 보류
- 하 과장 코웃음, 이 과장 "뭐 그렇지" 표정, 윤 사원 눈치

**⑥ [수요일, 오후 6:03] 김 대리 - 칼퇴**
- 하 과장 제일 먼저 나감 (인사 없음, 의자도 안 밀음)
- 이 과장: "수고하세요" (힘없음)
- 정 대리: "팀장님, 수고하셨습니다" (억지는 아님)
- 윤 사원: "수고하셨습니다" (기계적, 정 대리 따라)
- **팀장+김 대리 둘만 남음**
- 팀장이 뭔가 말하려다 멈춤
- **"…내일 보자."** → **"예. 내일 뵙겠습니다."**
- 형광등: 여섯 개 책상에 아무도 없는데 불만 환함
- "지직, 하는 소리. 석 달 동안 매일 들었는데 처음 듣는 것 같았다."

**⑦ [수요일, 오후 7:08] 김 대리 - 퇴근길**
- 지하철 (평소 새벽 퇴근 → 7시에 나오니 어색)
- "지난 3개월 동안 외딴섬에서 일한다는 느낌이었는데"
- 새벽 퇴근이 더 편했다 (법인 카드 택시, 사람 없음)
- GitHub 잔디 텅텅 빔, **마지막 커밋 아홉 달 전**
- 전 회사: 퇴근 후 코드, 블로그, 오픈소스 → 지금: 침대, 유튜브, 기억 안 남
- 이직 고민: **"1년도 안 됐다. 열 달."**, **"3년은 되어야 한다고"**, "지난 3개월은 거의 3년 일한 것 같은데"
- 원룸: 편의점 지나고, 골목 꺾고, 계단 올라감
- **"시간이 8시 밖에 되지 않았지만 씻지도 않고 다시 불을 끄고 침대에 누웠다."**
- **마무리: "내일은 온다. 준비 같은 건 상관없이."** (EP1 엔딩 호응)

#### EP2-2에서 확립된 핵심 설정

### 에피소드 3-1: 균열의 시작 ✅

**② 김 대리 - 강남역 근처 배회**

## EP3/EP4 작성 시 필수 고려 사항

### 확정된 사실 (변경 불가)

1. **하 과장은 일요일에 안 나왔다** — EP3에서 하 과장의 불만 포인트는 "나오라고 전화한 것 자체"이지 "일요일에 나왔는데 보상이 없다"가 아님
2. **부서장의 약속은 처음부터 모호했다** — "넉넉하게 해줄게요", "보답" (EP2-1 확정 텍스트). "성과급", "유급 휴가"라는 단어는 부서장 입에서 나온 적 없음
3. **김 대리 입사 열 달** — EP3 시점에서는 약 1년 (두 달 후)
4. **김 대리 마지막 커밋 아홉 달 전** — EP3 시점에서는 열한 달 전
5. **릴리즈 시점**: 화요일 저녁 7시, EP2-2는 그 다음 날 수요일
6. **팀장의 대답 안 하는 습관**: 말 안 통하면 침묵. EP3 하 과장 대면에서도 유지

8. **법인 카드 택시**: 새벽 퇴근 시 이용 (EP2-2에서 확립)
9. **원룸 구조**: 편의점 5분, 골목 꺾기, 계단, 침대+책상+옷걸이

- 김 대리: 점심시간에 슬쩍 이직 사이트 봄. 죄책감 반, 해방감 반.

**③ [오전 11:00] 판매 실패 소식 전달**
- 팀장이 부서장실에서 내려옴. 표정이 아예 없음. 하얗게 질려 있음.
- **"다들 잠깐."** (세 번째 등장하는 이 패턴)

**④ [오후 2:00] 박 팀장 - 부서장실 호출**
- 부서장 분노 (하지만 여유로운 척 유지하면서 압박)
- "팀장님, 이게 팀장님이 자신 있다고 했던 결과예요?"
- 추가 개발 요구: 고객사 피드백 반영해서 고쳐라
- "이거 안 고치면 다음은 없어요."
- 팀장: "팀원들이 지쳐 있습니다. 석 달 야근하고, 성과급도 안 나왔고…"

**⑤ [오후 3:30] 김 대리 시점 - 사무실, 팀장 발표**
- 팀장 내려옴. 전체 공유: "추가 개발 들어간다."

  - "잠깐만요. 아니, 성과급도 안 나왔는데 추가 개발이요?"
  - "석 달 야근하고, 주말에도 나오고, 그 대가가 추가 개발이요?"

  - 다른 팀원들 굳어짐

**⑥ [오후 3:45] 김 대리 시점 - 팀장-하 과장 회의실 충돌 (관찰자 시점)**

- 팀장: 처음에는 달래다가 → 한계에 달함
- **팀장 반말 전환 (격앙):** **"니가 진짜 그렇게 열심히 했냐고"**
  - 이 한마디에 회의실 안 정적
  - 팀장 본인도 후회하는 표정
  - 하 과장: 표정이 굳음. 뭔가가 꺼짐.
- 김 대리가 보는 것: 회의실 유리 너머로 두 사람의 표정 변화
- 하 과장 나옴. 자리로 돌아가서 가방 쌈. **칼퇴.** 이날 이후로 매일 칼퇴.

  - **"연봉 덜 받더라도, 야근 더 하더라도, 의미 있는 일을 하고 싶습니다."**
  - 돈 문제가 아님. 고객도 없고 방향도 없는 일을 반복하는 게 문제.

  - 처음에는 "생각 좀 더 해봐" 식으로 나옴
  - 그러다 진심으로: **"내가 바꿔볼게. 조금만 기다려줘."**
  - 뭔가 계획이 있는 듯한 눈빛. 구체적으로 말하진 않음.
  - "이번 추가 개발 끝나면, 다음에는 제대로 된 프로젝트를 제안해볼 거야."
- 김 대리: 반신반의. 하지만 팀장 눈에서 처음으로 확신 비슷한 걸 봄.
- "…한 달만 더 보겠습니다." (완전한 약속은 아님)

**⑧ [새벽 1:30] 박 팀장 - 집 서재**
- 아내와 아이들 잠든 후.
- 노트북 앞에 앉아 기획서 작성.
- **"다이렉트 프로젝트"** — 직접 고객을 만나서 니즈를 파악하고, 자체적으로 기획-개발하는 프로젝트
  - 지금까지의 문제: 대행업체 거쳐서 요구사항 전달 → 실제 사용자 모름 → 엉뚱한 걸 만듦
  - 해결: 직접 고객 접점을 만들자. 작은 규모로 시작.
- 팀장의 동기: 김 대리한테 한 약속 + 하 과장 충돌의 충격 + 8년간의 답답함
- 스타트업 하던 때 떠올림: 그때는 내가 결정하면 됐다. 망해도 내 책임이었다.
- 기획서 다듬고 또 다듬음. 새벽 3시.
- 아내가 서재에 와서 "자야지" 함. "응, 금방 잘게."
- 잠들기 전 생각: 이게 승인되면 뭔가 바뀔 수 있을까.

**⑨ [며칠 뒤, 오후 4:00] 박 팀장 - 부서장실, 기획서 보고**
- 추가 개발 진행하면서 틈틈이 기획서 보완
- 부서장에게 보고: "이번 추가 개발 마무리되면, 이 프로젝트를 제안드리고 싶습니다."
- 부서장 반응:
  - 처음: 시큰둥. "이걸 왜?"
  - 기획서 넘기며 읽음. 점점 표정 변화.
  - "직접 고객 접점?" "자체 기획?"
  - 부서장 입장에서: 임원 승진에 새로운 성과가 필요함. 대행업체 통한 프로젝트는 실패. 새 카드가 필요.
- **승인.** 단, 조건: **"이번 추가 개발 먼저 깔끔하게 마무리하세요."**
- 팀장 처음으로 부서장실 나오면서 표정 밝음.
- 이것이 에피소드 3의 마지막 장면. 어둠 속의 한 줄기 빛.

### 핵심 포인트
- 판매 실패 → 추가 개발 → 하 과장 충돌 → 김 대리 퇴사 선언 → 기획서 승인
- 하 과장 변화의 **결정적 순간**: "니가 진짜 그렇게 열심히 했냐고"
- 김 대리 퇴사 → 팀장의 각성 촉매
- 기획서 승인 = 에피소드 4의 "희망" 파트 세팅
- 분량 1.5~2배 (약 5,000~6,000자)

## 에피소드 4: 결말 (희망 → 좌절 → 퇴사) (미작성)

**시점:** 에피소드 3 이후 석 달 뒤 (다이렉트 프로젝트 두 달째 진행 중) / **분량:** 기준 4배 (약 12,000~16,000자)

### 전체 구조

```
[전반부: 희망] — 느긋하고 따뜻한 톤
  장면 1~2: 달라진 분위기, 다이렉트 프로젝트 잘 진행 중
    ↓
[전환점: 좌절] — 급격한 톤 전환
  장면 3: 부서장 호출, 전면 재개발 지시, 다이렉트 보류
    ↓
[후반부: 퇴사와 남겨진 자들] — 빠른 시간 전개 (30분 내)
  장면 4~9: 팀장 퇴사 → 팀원들의 충격 → 열린 결말
```

### 장면 흐름

**① [오전 9:05] 박 팀장 - 사무실 (희망)**

- 다이렉트 프로젝트 두 달째. 직접 고객사를 만남. 처음으로 "누가 쓰는지" 앎.
- 팀원들 변화:
  - 김 대리: 코드 리뷰 부활. PR 올리면 팀장이 직접 리뷰. "이거 이렇게 하면 더 깔끔한데?" 같은 대화.
  - 정 대리: 밝아짐. "팀장님, 이거 이렇게 해봐도 돼요?" 질문 많아짐.
  - 윤 사원: 처음으로 자기 의견을 말함.
