> 인터뷰 일시: 2026-02-08
> 상태: 인터뷰 완료, 작성 대기

## 기본 정보

- **에피소드:** 2-2 (결과보고)
- **화자:** 김 대리 (1인칭)
- **장면 수:** 6개 (①~⑥)
- **목표 분량:** 약 6,500자
- **시간대:** 수요일 오전 10:40 ~ 저녁 7시쯤

## 감정 곡선

```
① 도입: 회의적 (기대 낮음, '어차피 그럴 거')
     ↓
② 전달: "그럼 그렇지" (충격 아닌 확인)
     ↓
③ 하 과장 반응: 담담한 관찰 ("믿고 싶은 대로 믿는 게 사람이지")
     ↓
④ 추가 통보: 감정 미세한 누적
     ↓
⑤ 칼퇴: 다 나간 빈 사무실, 팀장의 미안함
     ↓
⑥ 퇴근길: 공허한 피로
```

**핵심:** 김 대리는 이 시점 전체에서 **충격받지 않는다.** "그럼 그렇지"가 기저. 분노도 슬픔도 아닌 공허함.

---

## 장면별 핵심 정리

### ① [오전 10:40쯤] 사무실 — 팀장 돌아옴

- 박 팀장이 부서장실에서 내려온 것처럼 보임 (실제는 흡연실 거쳐 올라옴)
- **담배 냄새 미세하게** — "희미하게 담배 냄새가 났다" 정도. 의아하되 의미 부여 없이 패싱
- 팀장 표정 묘함. 눈이 멍함.
- 김 대리의 출발 감정: **회의적**. 보상 기대 낮음.
- 도입 방식: 박 팀장 시점 마지막("자리를 향해 걸었다")에서 **바로 이어지게** — 팀장이 자리로 걸어오는 걸 김 대리 시점으로 받음

### ② 팀장 전달

- 좋은 소식 먼저: "부서장님이 잘했다고, 대행업체 반응도 좋다고"
- 나쁜 소식: "챙겨준다는 게 성과급이었대. 판매 성과 나와야 한대."
- **김 대리 반응:** 충격이 아닌 확인. "그럼 그렇지."
- **시점 특성:** 자기 감정보다 **다른 팀원들의 반응을 관찰하는 데 집중**
- 타인 반응에 대한 내면: "믿고 싶은 대로 믿는 게 사람이지" — 냉소적 공감

### ③ 하 과장 반응 (긴장 고조, 아직 폭발은 아님)

- 하 과장: "토요일에 부서장님이 직접 넉넉하게 챙겨준다고 하셨잖아요."
- "저도 개인 일정 취소하고 일요일에 나왔어요."
- "그랬는데 언제인지 모른다고요?"
- 팀장 대답 못 함. 침묵.
- 하 과장 더 따지려다 **참음**: "…됐어요." (이 한마디로 마무리)
- **김 대리의 관찰 방식:** 감정 없이 담담하게 기록. 하 과장의 표정, 행동을 건조하게 나열.
- EP3 폭발의 복선.

### ④ 추가 통보: 휴가 보류 + 수정 대기

- "대행업체 수정 요청 올 수 있으니 대기하래."
- "휴가는 수정 끝나고 쓰래."
- 석 달 고생한 대가 = 보상 미정 + 휴가 보류 + 수정 대기
- 하 과장 코웃음. 이 과장 무반응 ("뭐 그렇지" 표정). 윤 사원 눈치만 봄.

### ⑤ 칼퇴 — "오늘은 일찍 가"

- 팀장: "오늘은 부서장님이 칼퇴하라고 하셨어. 일찍 가."
- 여섯 시. 다들 일어남.
- **퇴근 순서:**
  - 하 과장 제일 먼저 나감. 가방 챙기고 인사 안 함.
  - 이 과장: 조용히 나감.
  - 정 대리: "팀장님, 수고하셨습니다." (아직 긍정적)
  - 윤 사원: "수고하셨습니다." 기계적으로.
  - **김 대리: 가장 나중에 나감.**
- **팀장과 둘이 남는 순간:**
  - 팀장이 "...미안해" 성격의 한마디를 하려다 멈춤
  - 둘 사이의 미세한 교환
- **환경 묘사:** 다 나간 빈 사무실, **형광등 켜져 있음** — 아무도 없는데 불만 환한 공간

### ⑥ [저녁 7시쯤] 퇴근길

- 평소보다 일찍 나왔는데 허전함
- 석 달 동안 매일 밤 10~11시에 나갔는데, 7시에 나오니 어색함
- 지하철 (원룸 자취)
- **개발자 정체성 자각:**
  - GitHub 잔디 텅텅 빔 (개인 계정)
  - 예전에는 퇴근 후 사이드 프로젝트도 했는데, 지금은 침대에 누워 유튜브 보다 잠듦
  - 이직해야 하나. 1년도 안 됐는데.
- **끝 감정: 공허한 피로**
- **마무리:** "내일도 출근해야 한다. 대기하라고 했다. 수정 요청이 올 수 있다고. 성과급은 언제인지 모른다고. 석 달 고생한 대가가 그거다."
- EP1 엔딩 호응: "내일은 온다. 준비 같은 건 상관없이."

## 시점 전환 & 연결

| 포인트 | 처리 방식 |
|--------|----------|
| 박 팀장 → 김 대리 전환 | 바로 이어지게. 팀장이 자리로 걸어오는 걸 김 대리 시점으로 받음 |
| 담배 냄새 | 미세하게 처리. "희미하게 냄새가 났다" 정도. 의미 부여 없이 스쳐가는 감각 |
| EP2-1과의 연결 | 팀원들의 보상 기대(보너스, 유급 휴가)가 이미 세팅된 상태 |
| EP3과의 복선 | 하 과장의 "…됐어요." = 참음. 폭발 전 긴장 축적 |

## 김 대리 시점 특성

- **관찰자 모드**: 자기 감정보다 타인 반응 관찰에 집중
- **회의적 톤**: "그럼 그렇지", "믿고 싶은 대로 믿는 게 사람이지"
- **개발자 정체성**: 퇴근길에서 기술 성장 정체 자각 (GitHub, 사이드 프로젝트, 코드 리뷰 부재)
- **공허한 피로**: 분노도 슬픔도 아닌 공허함이 기저

## 참조: 3단 기대 구조 (EP2-2 전체)

| 주체 | 기대한 것 | 실제 |
|------|----------|------|
| 팀원들 | 보너스 + 유급 휴가 + 밀린 연차(당연) | 성과급(언제?) + 휴가 보류 + 대기 |
| 박 팀장 | 보너스 + 유급 휴가 (연차는 안 물어봄) | 성과급 리프레이밍 + 유급 휴가 무시 + 연차 차단 |
| 부서장 | (처음부터 구체적 약속 없었음) | "성과가 있어야 성과급" — plausible deniability |

- 감정 직접 표현 금지. 관찰과 행동으로.
- 장면 묘사 15~20% (문학적 강화 방향)
- 대화 사이 여백 — 핵심 대사 전후에만
- 시간 표기: `### [수요일, 오전/오후 시:분] 김 대리 - 장소`
- "석 달" 통일, "부서장님" 통일
