# 허공을 미는 사람들 — 문체 가이드 (Claude Code Skill)

> 이 문서는 소설 "허공을 미는 사람들"의 문체를 일관되게 유지하기 위한 작성 규칙이다.
> 새 에피소드 작성, 수정, 리뷰 시 이 가이드를 참조한다.

---

## 1. 서술 기본 구조

### 시제: 과거형 기본 + 현재형 내면 독백 교차

- **서술/행동/대화**: 과거형 (`~했다`, `~였다`)
- **내면 독백/감정 반응**: 현재형 (`~인가`, `~거지`, `~싶다`)
- 두 시제의 교차가 건조한 표면 아래 감정을 드러내는 핵심 장치

```
# 좋은 예
나는 대답하지 않았다. 요즘 이상한 습관이 생겼다.
상대방의 말이 안 통하면 가끔 그 말을 듣고도 대답하지 않는다.  ← 과거형 서술

불안하다. 왜 갑자기 그 얘기를 꺼내는 거지.  ← 현재형 내면 독백
```

### 시점: 1인칭 제한 시점

- 박 팀장 시점 / 김 대리 시점 교차 (에피소드별 또는 장면별)
- 해당 화자가 **직접 보고 듣고 느낀 것만** 서술
- 다른 인물의 내면은 추측으로만 표현 (`~인 것 같았다`, `~인지 모르겠다`)

## 2. 문장 리듬

### 2.1 짧은 문장 연쇄로 긴장감 조성

압박, 충격, 감정 고조 장면에서 문장을 짧게 끊는다.

```
심장이 멈춘 것 같았다.
"사무실에 아무도 없더라고요?"
"…"
입이 열리지 않았다.
```

### 2.2 체언 종결 + 코멘트

핵심 단어를 문장 앞에 꺼내고, 바로 뒤에 화자의 반응을 붙인다.

```
푸시. 어딜 더 밀어야 할까.
헌신. 또 그런 단어다.
유급 휴가. 성과급. 넉넉하게. 그 말들이 귀에 맴돌았다.
```

### 2.3 단답 반복

기계적 응답의 반복으로 무력감·자동화된 복종을 표현한다.

```
"예."
"예."
"예."
기계적으로 대답했다. 머릿속이 하얘졌다.
```

### 2.4 문장 종결 다양화

`~했다` 단조로움을 피한다. 아래 종결 유형을 섞어 쓴다.

| 유형 | 예시 |
|------|------|
| 과거형 서술 | `나는 후다닥 부서장실을 나왔다.` |
| 현재형 독백 | `불안하다.` |
| 체언 종결 | `익숙한 톤이다.` |
| 의문형 독백 | `나는 왜 못 하지.` |
| 부정형 | `신경 쓸 여유가 없었다.` |
| 생략/여운 | `그게 전부였다.` |

## 3. 감정 표현 원칙

### 핵심: 감정을 직접 말하지 않되, 읽는 사람이 느끼게 한다

### 3.1 사물/환경으로 심리 투영

감정을 설명하지 않고, 주변 사물의 상태로 비춘다.

```
# 좋은 예
국이 완전히 식었다. 김이 더 이상 나지 않았다.
먹어야 하는데. 손이 움직이지 않았다.

# 나쁜 예
나는 너무 슬프고 무력했다. 아무것도 할 수 없었다.
```

### 3.2 행동/신체 반응으로 감정 전달

```
손이 떨렸다.
다리에 힘이 풀렸다.
복도 벽에 잠시 기대섰다.
입꼬리가 올라가지 않았다.
```

### 3.3 타인의 반응으로 화자 상태 간접 표현

```
# 딸이 아빠 표정을 보고 말없이 돌아감 → 화자의 일그러진 표정을 독자가 추론
딸이 나를 쳐다봤다. 뭔가 말하려는 것 같았다.
하지만 말이 나오지 않는 것 같았다. 내 표정을 보고 있었다.
딸이 문을 닫았다. 뛰어가지 않고 천천히 걸어갔다.
```

### 3.4 금지 사항

- 감정 형용사 직접 나열 금지 (`슬펐다`, `화가 치밀었다`, `감동적이었다`)
- 자기 연민 톤 금지 (`나는 왜 이렇게 불행한가`, `이 세상에 나만 힘든 것 같았다`)
- 감정에 대한 반복 설명 금지 (같은 감정을 다른 문장으로 2번 이상 풀지 않는다)

## 4. 대화 규칙

### 4.1 대화와 서술의 온도차

부서장 등 상위자의 대사는 표면적으로 정중하지만, 화자의 서술이 즉시 이면을 해석한다.

```
"팀장님, 별일 없지요?"
내가 인사하기도 전에 부서장님은 먼저 안부를 묻는다.
프로젝트 일정의 특이사항을 묻는 것이다.
```

### 4.2 조직 언어의 번역

조직 내 완곡 표현을 화자가 속으로 번역하는 구조를 활용한다.

```
"팀원들이 지금보다 훨씬 더 헌신한다면 분명히 다른 부분까지 추가로 해낼 수 있을 거라 봅니다."
→ (화자 내면) 헌신. 또 그런 단어다.
```

### 4.3 대화 자연스러움

- 한국 직장 경어 체계 엄격 준수: 부서장에게 `~습니다`, 팀원 간 `~요/~해`
- 격앙 시 비속어 허용하되 최소한으로 (`씨발` — 하 과장 등 특정 캐릭터만)
- 대화 중 `…` 침묵 활용: 말을 잇지 못하는 순간, 감정 억제 순간

## 5. 반복 모티프

에피소드를 관통하며 무력감의 질감을 만드는 반복 장치들.

| 모티프 | 의미 | 사용 예시 |
|--------|------|-----------|
| 시계 초침 소리 `똑. 똑. 똑.` | 멈출 수 없는 시간, 피할 수 없는 내일 | 장면 말미 여운 |
| 식어가는 밥/국 | 돌봄이 닿지 못하는 거리 | 처가댁, 집 장면 |
| 흡연실 (피우지 않는 담배) | 도피처 없는 도피 | 박 팀장 습관 |
| 부서장의 안경 너머 시선 | 권력의 응시, 판단의 순간 | 부서장실 장면 |
| `예.` 반복 | 자동화된 복종 | 전화 통화, 면담 |
| 아이들 손/얼굴 | 지키고 싶지만 닿지 못하는 것 | 집/처가댁 장면 |

**주의:** 같은 모티프를 한 에피소드 안에서 3회 이상 쓰지 않는다. 패턴화되면 예측 가능해진다.

## 8. 어휘/표현 규칙

### 사용 금지

| 금지 표현 | 이유 | 대체 |
|-----------|------|------|
| `애들이 잤다` | 구어적, 톤 불일치 | `아이들은 잠들어 있었다` |
| `애들 자는 걸 본 게` | 위와 동일 | `잠든 얼굴을 본 게` |
| `한 달` (프로젝트 기간) | 설정 불일치 | `석 달` |
| `3개월`, `세 달` | 표기 통일 | `석 달` |
| 과도한 비유/문학적 수사 | 톤 불일치 | 최대한 건조하게 |
| `근데` 과다 사용 | 반복감 | 적절히 분산 |
| `한숨을 쉬었다` 반복 | 진부화 | 다양한 신체 반응으로 대체 |
| `모니터를 쳐다봤다` 반복 | 위와 동일 | 변형 |

### 숫자 표기

- 기간: 한글 (`석 달`, `두 달`, `일주일`)
- 시각: 아라비아 숫자 (`[오후 3:24]`)
- 나이: 한글 + 살 (`여섯 살`, `네 살`) — 서술 시 / 아라비아 + 세 (`41세`) — 설정 문서
- 금액: 아라비아 숫자 + 한글 단위 (`300억`)
- 인원: 한글 (`여섯 명`)

## 9. 피해야 할 톤

| 유형 | 예시 | 문제 |
|------|------|------|
| 자기 연민 과잉 | `이 세상에서 가장 불행한 사람은 나일 것이다` | 독자가 동정 대신 거부감 느낌 |
| 교훈적 마무리 | `그래서 직장이란 이런 곳인 것이다` | 독자가 스스로 해석할 여지 제거 |
| 과잉 드라마 | `눈물이 주르르 흘렀다. 세상이 무너지는 것 같았다` | 이 작품의 톤은 "울컥했다. 참았다." |
| 희망 과잉 | `내일은 분명 나아질 것이다` | 희망은 항상 불안과 함께 (`~나. ~나.` 의문형) |

## 10. 희망/불안 표현 패턴

이 작품에서 희망은 단독으로 존재하지 않는다. 항상 불안과 짝을 이룬다.

```
# 희망회로 패턴
속으로 희망회로를 돌려본다.

'끝나면 휴가 간다고 했으니까.'
'성과급 나오면 아이들 데리고 여행 가자.'
'며칠만 더 버티면 된다.'

그런데 그 불안감은 가시지 않았다.
부서장님 말을 믿어도 되나. 끝나면 정말 쉴 수 있나.
```

- 희망 → 의문형 불안 (`~나`, `~되나`, `~건가`)으로 전환
- 같은 에피소드 내 2회까지 허용, 3회부터 패턴 예측 가능

## 11. 기술 디테일 수준

개발 관련 디테일은 **개발자 독자가 고개를 끄덕일 정도**로, 비개발자도 맥락은 이해할 수 있게.

```
# 좋은 예 — 구체적이되 설명적
기능 하나를 추가하는데, 비슷한 코드가 이미 세 군데 있었다.
공통 모듈로 빼면 되는 건데, 그러려면 기존 코드를 건드려야 한다.
기존 코드를 건드리면 테스트를 돌려야 한다. 테스트가 없다.

# 나쁜 예 — 너무 기술적
AbstractFactoryPattern을 적용해서 DI 컨테이너를 리팩토링해야 했다.

# 나쁜 예 — 너무 모호
코드가 엉망이었다. 고치기 힘들었다.
```

## 12. 장면 마무리 패턴

각 장면의 마지막은 아래 유형 중 하나로 닫는다.

| 유형 | 예시 | 효과 |
|------|------|------|
| 시간 예고 | `여섯 시간 후면 알람이 울린다.` | 피할 수 없는 내일 |
| 사물 정지 | `국이 완전히 식었다.` | 여운, 정서 투영 |
| 질문 | `뭐라고 말하지.` | 미해결 긴장 |
| 감각 | `똑. 똑. 똑.` | 공허한 반복 |
| 대비 | `조금은 괜찮았다.` (직전 고통 후) | 쓸쓸한 위안 |

## 13. 문학적 강화 방향 (에피소드 2-2~)

> 에피소드 1, 2-1은 웹소설에 가까운 톤이었다. 에피소드 2-2부터는 기존 건조한 톤을 유지하되,
> **문학적 깊이를 조금씩 더한다.** 급격한 변화가 아니라 방향성이다.

### 13.1 장면/환경 묘사 강화

기존 에피소드에서는 대화와 내면 독백 위주로 진행되었다. 이제는 **공간과 사물이 인물의 심리를 대신 말하게** 한다.

```
# 기존 스타일 (에피소드 1~2-1)
부서장실 문을 열기 전에 심호흡을 한 번 했다. 습관이다.

# 강화 방향 (에피소드 2-2~)
부서장실 문 앞에 섰다. 복도 형광등이 미세하게 깜빡거렸다.
그 떨림이 내 것인지 형광등 것인지 모르겠다.
심호흡을 한 번 했다. 습관이다.
```

**원칙:**
- 묘사를 추가하되, **매 문단마다 넣지 않는다.** 장면 전환, 감정 전환점, 마무리에 집중 배치.
- 묘사는 항상 **심리와 연결**된다. 장식적 묘사 금지.
- 전체 분량 대비 묘사 비율: **기존 5~10% → 15~20%로 확대.** 그 이상은 과잉.

### 13.2 대화 사이 여백 확대

기존에는 대사가 빠르게 주고받았다. 이제는 대화 사이에 **침묵의 질감**을 넣는다.

```
# 기존 스타일
"성과급은요?"
"판매 성과 나와야 해."
"…그게 되나요?"

# 강화 방향
"성과급은요?"

팀장님이 입술을 열었다가 다시 닫았다. 책상 위 볼펜을 한 번 만졌다.
볼펜이 소리 없이 굴러갔다.

"판매 성과 나와야 해."

아무도 말하지 않았다. 에어컨 소리만 들렸다.

"…그게 되나요?"
```

**원칙:**
- 모든 대사 사이에 여백을 넣는 것이 아니다. **핵심 대사, 감정 전환 대사** 전후에만.
- 여백의 내용: 신체 반응(시선, 손, 호흡), 사물 접촉, 공간의 소리, 짧은 내면 반응.
- 여백 없는 빠른 대화도 리듬상 필요하다. **강약 조절**이 핵심.

### 13.3 장면 마무리의 깊이

기존 장면 마무리 패턴(12번 참조)에 더해, 마무리에 **한 호흡 더**를 준다.

```
# 기존 스타일
내일은 온다. 준비 같은 건 상관없이.

# 강화 방향
거실 시계 초침 소리가 들렸다. 똑. 똑. 똑.
창밖은 어두웠다. 가로등 불빛이 커튼 사이로 가늘게 들어왔다.

내일은 온다. 준비 같은 건 상관없이.
```

**원칙:**
- 마무리 직전에 **환경 묘사 1~2문장**을 배치하여 감정을 가라앉힌 후 마지막 문장.
- 마지막 문장 자체는 기존처럼 **짧고 건조하게** 유지.

### 13.4 변하지 않는 것

문학적 강화에도 **절대 변하지 않는 원칙**:

- 건조한 톤. 감정을 직접 말하지 않는다.
- 과거형 서술 + 현재형 내면 독백 교차.
- 자기 연민 금지. 과잉 드라마 금지.
- 이 소설의 힘은 **절제**에서 나온다. 묘사를 추가하되 절제를 잃지 않는다.
- 비유는 여전히 최소한으로. 직유 1~2개/장면 이하.
